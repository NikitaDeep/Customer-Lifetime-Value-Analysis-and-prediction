# Customer-Lifetime-Value-Analysis-and-prediction
A SQL-based analytics project designed for eCommerce data to uncover insights on customer lifetime value (CLTV), churn prediction, product profitability, and trend forecasting. It includes advanced SQL queries for segmentation, refund impact, and retention strategy—turning raw transactional data into meaningful business intelligence



Project Overview

SQL-based analytics project for eCommerce data focused on understanding Customer Lifetime Value (CLTV), churn prediction, product profitability, and trend forecasting.

Purpose

To analyze customer behavior, identify churn risks, and improve profitability through data-driven insights.

Key Findings

•	30% of customers contributed 65% of total revenue

•	Refunds reduced profit margins by 8–10%

•	38% of customers were identified as “At Risk”

•	Top 5 SKUs accounted for 45% of sales

Tools & Techniques

•	SQL (MySQL): Data extraction, transformation, analysis

•	Power BI / Excel: Visualization & dashboarding

•	RFM Segmentation: Customer classification and churn analysis

Outcome

Transformed raw transactional data into actionable business intelligence, helping optimize retention, profitability, and marketing strategies.
